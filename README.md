# PhishBuster – Cybersecurity Awareness Game

An interactive **Streamlit** web app that helps users learn to recognize phishing emails through quick rounds and instant feedback.

## How It Works
Each round presents a simulated email.  
You choose **Phishing** or **Safe** → get immediate feedback and learn the clues that reveal phishing attempts.

## Tech Stack
- Python  
- Streamlit  
- Pandas  

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
